To begin using the Material Design 3 Expressive component suite, install the core packages and configure your React environment.

## Installing Packages

### Using pnpm (Recommended)

```bash
pnpm add @bug-on/md3-react motion
```

### Using npm

```bash
npm install @bug-on/md3-react motion
```

### Using yarn

```bash
yarn add @bug-on/md3-react motion
```

## Peer Dependencies

Ensure your project has compatible versions of the following:

| Package | Version |
|---------|---------|
| `react` | `^18.0.0` or `^19.0.0` |
| `motion` | `>=12.0.0` |
| `tailwindcss` | `^4.0.0` |

## Typography & Icons

The library uses **Google Sans Flex** and **Material Symbols**. Add the following to your main CSS file (`globals.css` or `App.css`):

```css
@import "@bug-on/md3-react/typography.css";
@import "@bug-on/md3-react/material-symbols-cdn.css";
```

Alternatively, use the preconnect component to optimize loading performance:

```tsx
import { MaterialSymbolsPreconnect } from "@bug-on/md3-react";

export default function RootLayout({ children }) {
  return (
    <html>
      <head>
        <MaterialSymbolsPreconnect variants={["outlined", "rounded", "sharp"]} />
      </head>
      <body>{children}</body>
    </html>
  );
}
```

## Provider Setup

Wrap your application with `MD3ThemeProvider` — **a single provider** that handles theming, typography, animations, and optionally snackbar notifications:

```tsx
import { MD3ThemeProvider } from "@bug-on/md3-react";

export default function RootLayout({ children }) {
  return (
    <MD3ThemeProvider
      sourceColor="#6750A4"
      defaultMode="light"
      persistToLocalStorage
    >
      {children}
    </MD3ThemeProvider>
  );
}
```

To enable the built-in Snackbar queue, add `enableSnackbar`:

```tsx
<MD3ThemeProvider
  sourceColor="#6750A4"
  defaultMode="light"
  persistToLocalStorage
  enableSnackbar
>
  {children}
</MD3ThemeProvider>
```
