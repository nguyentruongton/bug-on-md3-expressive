To begin using the Material Design 3 Expressive component suite, follow these steps to install the core package and configure your environment.

## 1. Installing Packages

Install the main library and its required animation peer dependency (`motion`).

### Using pnpm (Recommended)

```bash
pnpm add @bug-on/md3-react motion
```

### Using npm

```bash
npm install @bug-on/md3-react motion
```

---

## 2. Tailwind CSS Configuration

Bug On MD3 Expressive provides a dedicated plugin that injects design tokens, elevation utilities, and Material Symbols support into Tailwind.

### For Tailwind CSS v4 (CSS-first)
Import the base styles and register the plugin directly in your main CSS file (e.g., `globals.css` or `app.css`):

```css
@import "tailwindcss";

/* 1. Import MD3 Tokens & Base Styles */
@import "@bug-on/md3-react/index.css";

/* 2. Register the MD3 Plugin */
@plugin "@bug-on/md3-react/plugin";
```

### For Tailwind CSS v3 (JavaScript-first)
Add the plugin to your `tailwind.config.ts` or `tailwind.config.js`:

```ts
import md3Plugin from "@bug-on/md3-react/plugin";

export default {
  content: [
    "./node_modules/@bug-on/md3-react/dist/**/*.{js,mjs}",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  plugins: [
    md3Plugin,
  ],
};
```

---

## 3. Typography & Icons

The library uses **Google Sans Flex** for typography and **Material Symbols** for icons. Ensure these assets are loaded in your main entry point:

```css
/* Add these to your globals.css */
@import "@bug-on/md3-react/typography.css";
@import "@bug-on/md3-react/material-symbols-cdn.css";
```

> **Tip:** You can also use the `MaterialSymbolsPreconnect` component in your Next.js `RootLayout` to optimize icon font loading performance.

---

## 4. Provider Setup

Wrap your application with `MD3ThemeProvider`. This single provider handles the Material You dynamic color system, theme state, and optional snackbar notifications.

### For Next.js (App Router)
Since it uses React context, ensure the provider is in a **Client Component**.

```tsx
"use client";

import { MD3ThemeProvider } from "@bug-on/md3-react";

export default function Providers({ children }) {
  return (
    <MD3ThemeProvider
      sourceColor="#6750A4" // Your primary brand color
      defaultMode="light"   // 'light', 'dark', or 'system'
      persistToLocalStorage
    >
      {children}
    </MD3ThemeProvider>
  );
}
```

---

## 5. Usage

Now you can start using the components with full MD3 Expressive styling and animations!

```tsx
import { Button, Icon } from "@bug-on/md3-react";

export default function MyComponent() {
  return (
    <Button variant="filled">
      <Icon name="add" />
      Get Started
    </Button>
  );
}
```
