To begin using the Material Design 3 Expressive component suite, you need to install the core packages and configure your React environment.

## Installing Packages

The library is modularized into specialized packages. We recommend installing at least `@bug-on/md3-react` for core UI components.

### Using pnpm (Recommended)

```bash
pnpm add @bug-on/md3-react framer-motion clsx tailwind-merge
```

### Using npm

```bash
npm install @bug-on/md3-react framer-motion clsx tailwind-merge
```

### Using yarn

```bash
yarn add @bug-on/md3-react framer-motion clsx tailwind-merge
```

## Peer Dependencies

Ensure your project has compatible versions of the following libraries:

- **React**: ^18.0.0 or ^19.0.0
- **Framer Motion**: ^11.0.0 (required for animations)
- **Tailwind CSS**: ^3.4.0 or ^4.0.0

## Typography & Icons Configuration

The library utilizes **Google Sans (or equivalent)** and **Material Symbols**. Add the following imports to your main CSS file (e.g., `globals.css` or `App.css`):

```css
/* Import library styles */
@import "@bug-on/md3-react/typography.css";
@import "@bug-on/md3-react/material-symbols-cdn.css";
```

Alternatively, use the preconnect component to optimize loading performance:

```tsx
import { MaterialSymbolsPreconnect } from "@bug-on/md3-react";

export default function RootLayout({ children }) {
  return (
    <html>
      <head>
        <MaterialSymbolsPreconnect variants={["outlined", "rounded", "sharp"]} />
      </head>
      <body>{children}</body>
    </html>
  );
}
```
