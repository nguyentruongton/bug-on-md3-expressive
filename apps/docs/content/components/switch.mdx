Switches are the most efficient way to allow users to toggle individual settings. The Bug On MD3 Expressive Switch emphasizes interactivity through fluid thumb motion and responsive state layers.

## Labeled Switches

Use the `label` prop to automatically associate descriptive text with the switch. This improves accessibility by allowing users to click the label to toggle the state.

<ComponentPreview name="switch-labeled" />

## Icons

The MD3 Switch supports displaying icons inside the thumb to provide additional visual context. You can choose to display icons in both states or only when selected.

<ComponentPreview name="switch-icons" />

## States

Switches support a `disabled` state. When disabled, the switch blocks interaction and is rendered with reduced opacity according to Material Design 3 specifications.

<ComponentPreview name="switch-states" />

## Usage

```tsx
import { Switch } from "@bug-on/md3-react";

// Basic
<Switch checked={isOn} onCheckedChange={setIsOn} />

// With Label
<Switch checked={isOn} onCheckedChange={setIsOn} label="Wi-Fi" />

// With Icons
<Switch 
  checked={isOn} 
  onCheckedChange={setIsOn} 
  icons 
  thumbContent={isOn ? <CheckIcon /> : <CloseIcon />} 
/>
```

## Accessibility

- Utilizes the standard ARIA Switch pattern (`role="switch"`).
- State is communicated via the `aria-checked` attribute ("true" or "false").
- Full keyboard support: **Space** or **Enter** to toggle.
- When using `label`, the switch is wrapped in a `<label>` tag and linked via `id`.
- Focus rings are only visible during keyboard navigation (`focus-visible`).

## API Reference

### Switch

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | **Required**. Controlled checked state. |
| `onCheckedChange` | `(checked: boolean) => void` | — | **Required**. Callback fired when the state changes. |
| `disabled` | `boolean` | `false` | Disables interaction. |
| `label` | `string` | — | Text label displayed next to the switch. |
| `ariaLabel` | `string` | — | Accessible label for screen readers (use when `label` is omitted). |
| `icons` | `boolean` | `false` | Displays icons inside the thumb in both states. |
| `showOnlySelectedIcon` | `boolean` | `false` | Only displays the icon when the switch is in the 'on' state. |
| `thumbContent` | `ReactNode` | — | Icon (SVG) to be displayed inside the thumb. |
| `className` | `string` | — | Custom CSS class for the wrapper. |
