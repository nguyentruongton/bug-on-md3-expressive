Floating action buttons (FABs) represent the most important action on a screen. Expressive FABs use shape morphing on tap, elevation, and Ripple effects to communicate interaction.

## Sizes

MD3 Expressive defines three FAB sizes. The small FAB has been **deprecated** in the Expressive update and is not supported.

<ComponentPreview name="fab-sizes" />

| Size | Dimensions | Icon size | Border radius |
|------|-----------|-----------|---------------|
| Default | 56 Ă— 56dp | 24dp | 16dp |
| Medium | 80 Ă— 80dp | 28dp | 20dp |
| Large | 96 Ă— 96dp | 36dp | 28dp |

## Color Variants

FABs support six color styles from the MD3 color scheme. The `surface` variant has been deprecated in M3 Expressive.

<ComponentPreview name="fab-colors" />

## Lowered State

A lowered FAB sits at the same elevation as the content below it. Use `lowered` when the FAB is in a container that already provides separation from the background.

## Usage

```tsx
import { FAB, Icon } from "@bug-on/md3-react";

// Default FAB
<FAB icon={<Icon name="edit" />} aria-label="Edit" />

// Large FAB with tertiary color
<FAB size="large" colorStyle="tertiary" icon={<Icon name="edit" />} aria-label="Edit" />

// FAB with loading state
<FAB icon={<Icon name="edit" />} aria-label="Saving" loading />

// Hidden FAB (e.g., while scrolling down)
<FAB icon={<Icon name="edit" />} aria-label="Compose" visible={!isScrollingDown} />
```

Use `FABPosition` to pin the FAB to the bottom-right corner:

```tsx
import { FAB, FABPosition } from "@bug-on/md3-react";

<FABPosition bottom="1.5rem" right="1.5rem" safeArea>
  <FAB icon={<Icon name="edit" />} aria-label="Compose" />
</FABPosition>
```

## Accessibility

- The `aria-label` prop is **required**. FABs are icon-only buttons and must have a descriptive label for screen readers.
- When `loading={true}`, the component sets `aria-busy="true"` automatically and blocks click events.
- The default FAB (56dp) includes an invisible `48dp` touch target expander to satisfy WCAG 2.5.5 Target Size.
- Shape morphing and entrance/exit animations respect `prefers-reduced-motion` via Framer Motion.
- The FAB is keyboard accessible: focus shows a visible ring, and `Enter`/`Space` trigger the action.
